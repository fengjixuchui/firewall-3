firewall.exe v0.9
=================

Requirements:
-------------

  WinDivert 2.2 (included)
  https://www.reqrypt.org/windivert.html


Instruction
-----------

1. Edit config files

  -> in.txt (Inbound rules)
  -> out.txt (Outbound rules)

2. Disable Windows Firewall

3. Start firewall.exe
